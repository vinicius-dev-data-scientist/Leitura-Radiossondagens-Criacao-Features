\# 📊 Leitura de Radiossondagens e Criação de Features



Projeto para leitura, tratamento e extração de variáveis meteorológicas a partir de dados de radiossondagens.



\## 🚀 Objetivo

Processar dados atmosféricos e gerar features úteis para análise meteorológica e modelos de previsão.



\## ⚙️ Funcionalidades

\- Leitura de dados de radiossonda

\- Tratamento e limpeza dos dados

\- Cálculo de variáveis meteorológicas

\- Geração de features



\## 🛠️ Tecnologias utilizadas

\- Python

\- Pandas

\- MetPy



\## 📁 Estrutura do projeto

\- `leitura\_trata\_radiossonda.py`: leitura e tratamento dos dados

\- `calc\_feats\_metpy.py`: cálculo das variáveis meteorológicas

\- `feats\_indexes.csv`: saída com features calculadas
